# 🎨 AI Can't Guess This!

เกมปาร์ตี้แบบทีม: วาดรูปจากคำลับให้ **เพื่อนมนุษย์พอเดาออก** แต่ **AI เดาไม่ถูก** — หลอก AI สำเร็จรับคะแนน!

## วิธีรัน

```bash
npm install
npm start
```

ต้องมีไฟล์ `.env` ที่มี `OPENAI_API_KEY` (ถ้าไม่มี ระบบจะใช้คำสำรองในเครื่อง และปิดโหมด AI ทายภาพ)

| หน้า | URL |
|---|---|
| ผู้เล่น | http://localhost:3000/ |
| โฮสต์ (ควบคุมเกม + จอโปรเจกเตอร์) | http://localhost:3000/host |

รหัสผ่านโฮสต์เริ่มต้น: `admin123` (เปลี่ยนได้ด้วย env `HOST_PASSWORD`)

## กติกา

1. **Login** — ผู้เล่นเลือกทีม (สูงสุด 8 ทีม ทีมละกี่คนก็ได้) ใส่รหัสทีมและชื่อเล่น
2. **โฮสต์กดเริ่มรอบ** — ระบบสุ่มคำภาษาอังกฤษไม่ซ้ำกันให้แต่ละทีมผ่าน OpenAI
3. **วาดรูป** — สมาชิกทุกคนในทีมวาดบน canvas เดียวกันแบบ real-time เห็นเส้นของกันและกันทันที มี Undo / ล้าง / เลือกสี-ขนาดปากกา หมดเวลาระบบเก็บภาพอัตโนมัติ
4. **ทายภาพ** — ภาพถูกแจกแบบ Round-Robin (ทีมถัดไปได้ภาพ ไม่มีทีมไหนได้ภาพตัวเอง) พิมพ์คำตอบได้ทีมละ 1 ครั้ง ตอบภาษาไทยหรืออังกฤษก็ได้ (AI ช่วยตัดสินความหมาย)
5. **AI Guess** — GPT-4o vision ทายภาพทุกทีมพร้อมระดับความมั่นใจ
6. **คะแนน** — AI ทายผิด **+10** · มนุษย์ทายถูก **+5 โบนัส** · AI ทายถูก+มนุษย์ผิด **0**
7. **สรุปผล** — โชว์ทุกภาพ คำจริง คำตอบ AI/มนุษย์ ตารางอันดับสด และจำนวนครั้งที่หลอก AI สำเร็จ แล้วโฮสต์กดเริ่มรอบต่อไปได้เลย (คะแนนสะสมข้ามรอบ)

## Env ที่รองรับ

| ตัวแปร | ค่าเริ่มต้น | ความหมาย |
|---|---|---|
| `OPENAI_API_KEY` | — | คีย์ OpenAI (สุ่มคำ + vision + ตัดสินคำตอบ) |
| `OPENAI_MODEL` | `gpt-4o-mini` | โมเดลที่ใช้ |
| `HOST_PASSWORD` | `admin123` | รหัสหน้าโฮสต์ |
| `PORT` | `3000` | พอร์ตเซิร์ฟเวอร์ |
